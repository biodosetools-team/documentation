Biodose Tools

B Implementation details

B.1 Technologies

The Biodose Tools user interface is written in R Shiny (Chang et al. 2020) using Bootstrap 3 through the {shinydashboard} (Chang and Borges Ribeiro 2018) package, analyses are implemented in the R programming language (R Core Team 2020), with the resultant tables and plots rendered in HTML through JavaScript libraries. This is done by the browser of choice, or by an instance of QtWebKit if the app is run through RStudio.
